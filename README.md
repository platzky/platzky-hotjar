# Platzky Hotjar Plugin

## Overview

Platzky Hotjar is a plugin for the Platzky framework that allows you to set up hotjar easily.

## Installation

To install Platzky Hotjar, you can use pip:

```sh
pip install platzky-hotjar
```

### Usage

```json
"plugins": [
{
  "name": "hotjar",
  "config": {
    "/old-path": "/new-path"
  }
}
]
```
